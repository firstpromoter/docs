---
title: 'Lead Cancelled'
description: 'Fired when a lead or customer cancels their subscription.'
---

Your endpoint will receive this event when a lead's status changes to `cancelled`, either through a cancellation tracked by your billing integration or via the API.

## Payload

<ResponseField name="event" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="number">
      Unique ID of the event.
    </ResponseField>
    <ResponseField name="type" type="string">
      Always `lead_cancelled` for this event.
    </ResponseField>
    <ResponseField name="created_at" type="string">
      ISO 8601 timestamp of when the event occurred.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="number">ID of the lead.</ResponseField>
    <ResponseField name="status" type="string">Current state of the lead. Will be `cancelled` at time of this event.</ResponseField>
    <ResponseField name="email" type="string">Email address of the lead.</ResponseField>
    <ResponseField name="uid" type="string">External user ID passed when tracking the lead.</ResponseField>
    <ResponseField name="customer_since" type="string">Timestamp of when the lead became a customer. `null` if they cancelled before converting.</ResponseField>
    <ResponseField name="cancelled_at" type="string">Timestamp of when the cancellation occurred.</ResponseField>
    <ResponseField name="plan_name" type="string">Name of the plan the lead was subscribed to.</ResponseField>
    <ResponseField name="suspicion" type="string">Fraud suspicion level (e.g., `no_suspicion`).</ResponseField>
    <ResponseField name="username" type="string">Username of the lead, if provided.</ResponseField>
    <ResponseField name="website" type="string">Website of the lead, if provided.</ResponseField>
    <ResponseField name="created_at" type="string">Timestamp of when the lead was originally created.</ResponseField>
    <ResponseField name="split_promotion_id" type="number">ID of the split promotion, if applicable.</ResponseField>
    <ResponseField name="split_percentage_value" type="number">Percentage split value, if applicable.</ResponseField>
    <ResponseField name="promoter" type="object">
      The promoter who referred this lead. See [Lead Signup](/webhooks/lead-signup) for the full promoter object structure.
    </ResponseField>
    <ResponseField name="promotion" type="object">
      The specific promotion that generated this lead. See [Lead Signup](/webhooks/lead-signup) for the full promotion object structure.
    </ResponseField>
  </Expandable>
</ResponseField>
