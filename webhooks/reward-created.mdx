---
title: "Reward Created"
---


By setting up a webhook, your endpoint will receive an event when a reward is generated.

<Note>
This event is triggered anytime a reward is generated.
</Note>

## Response Paramerters

    <ResponseField name="id" type="string">
      The Id of the event
    </ResponseField>
