---
title: "Promoter Accepted"
---

By setting up a webhook, your endpoint will receive an event when a promoter is accepted.

## Response Parameters

<ResponseField name="Event" type="Event Object">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      The Id of the event
    </ResponseField>
    <ResponseField name="type" type="string">
      Indicates the webhook event type, such as `promoter_accepted`.
    </ResponseField>
    <ResponseField name="created_at" type="string">
      Timestamp of when the event occurred.
    </ResponseField>
  </Expandable>
</ResponseField>
